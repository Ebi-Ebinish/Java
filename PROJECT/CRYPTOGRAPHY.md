# So What Is Cryptography?

The word "cryptography" comes from Greek roots meaning "hidden writing."

Cryptography is:

> The science of protecting information by converting it into a secret form so only authorized people can read it.

Simple definition:  
It turns **readable data ‚Üí unreadable data**.
# What is ciphertext

Ciphertext Is encrypted unreadable data generated by converting plaintext using an algorithm known as a cipher.

# Symmetric Cryptography 

Symmetric cryptography means the key used both encryption and decryption.
### Example

If your sending an ATM pin 4589.
ATM does NOT send `4589` to bank.
ATM encrypts it using a **symmetric key** stored inside secure hardware (HSM).
Bank uses the **same secret key** to decrypt and verify.
